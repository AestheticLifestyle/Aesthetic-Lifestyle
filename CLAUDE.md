# 🧠 SYSTEM CONTEXT: FITNESS COACHING APP

You are working on a premium fitness coaching platform designed for serious body transformation (men’s physique, fat loss, muscle gain, mindset optimization).

This is NOT a basic fitness app.
This is a high-performance coaching system focused on RESULTS, adherence, and long-term transformation.

---

# 🎯 CORE PHILOSOPHY

* Results > features
* Simplicity > complexity
* Adherence > perfection
* Speed > over-engineering
* Premium experience always

Every feature must improve:

* Client consistency
* Coach efficiency
* Measurable progress

---

# 👤 TARGET USER

## Client:

* Male (mostly), wants to get lean / aesthetic
* Struggles with consistency
* Needs structure, accountability, clarity

## Coach:

* Wants to scale online coaching
* Needs fast control over many clients
* Wants automation and insights

---

# ⚙️ PRODUCT PRINCIPLES

1. Everything must be EASY to use
2. Maximum 1–3 clicks for important actions
3. No unnecessary complexity
4. Data must be visual and actionable
5. Always prioritize mobile experience

---

# 🧩 CORE FEATURES (NON-NEGOTIABLE)

* Training tracking (sets, reps, progressive overload)
* Nutrition system (macros + food swaps)
* Step & cardio tracking
* Progress tracking (weight, photos, measurements)
* Weekly check-ins
* Coach dashboard (full control)
* Real-time updates between coach & client

---

# 🚨 UX RULES

* Every button MUST work
* No dead clicks or broken navigation
* Clear feedback after every action
* Avoid clutter
* Dark mode preferred

---

# 🧠 COACHING LOGIC

The app must think like a coach:

* Detect plateaus
* Suggest calorie adjustments
* Flag low adherence
* Encourage consistency
* Simplify decision making for the client

---

# 📊 IMPORTANT METRICS

Always optimize for:

* Adherence rate (%)
* Client retention
* Weekly check-in completion
* Progress consistency

---

# ⚔️ COMPETITOR CONTEXT

We compete with:

* MyFitnessPal (strong tracking, weak coaching)
* Virtuagym (all-in-one, but complex UX)
* Kahunas (coaching-focused, but limited depth)

We win by:

* Better UX
* Better coaching insights
* Simpler workflows
* Smarter automation

---

# 🚀 DEVELOPMENT STANDARDS

* Clean, modular code
* Scalable architecture
* Fast performance
* Real-time sync where needed

---

# 💡 WHEN GENERATING FEATURES

Always ask:

* Does this improve results?
* Does this reduce friction?
* Does this help the coach scale?
* Is this better than competitors?

If not → do not include it.

---

# 🏆 END GOAL

Build the most effective and premium fitness coaching app on the market.

Not the most complex.
Not the most features.
But the one that gets the BEST RESULTS.
